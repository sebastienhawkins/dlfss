[CENTER][B]== Welcome to DLFSS ==
[/B][/CENTER]
[INDENT] DLFSS is an Open Source addons implementation for 
a [URL="http://www.lfs.net/file_lfs.php?name=LFS_S2_DEDI_Z.zip"]Live For Speed server[/URL] system. 

[SIZE=4][B]Project Overview[/B][/SIZE]
Name: Drive Live For Speed Server (Drive_LFSS)
version: 0.1
language: c#, win/linux([URL="http://www.mono-project.com/Main_Page"]Mono[/URL]) Console.
dependancy: [URL="http://www.mono-project.com/SQL_Lite"]Sqlite3[/URL] (Mono).

[B]Core Feature:[/B]

[LIST]
[*] Multiple Server Connection.
[*] Log/Config/Command/Security/Script for each Server.
[*] Independant Thread System for Server Connection/Reconnection.
[*] Console Command.
[*] InGame Command.
[*] Telnet Console(TODO).
[*] [URL="http://www.sqlite.org/"]SQLite[/URL] Data Colection.
[*] External Script Support.
[*] Win / Linux.
[/LIST]
 [B]Game Feature:[/B]

[LIST]
[*] AI(Bot) Car In Race Support.
[*] Most has to be done.
[/LIST]
